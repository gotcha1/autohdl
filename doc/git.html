Работа с гит репозиторием

При создании нового проекта всегда локально инициализируется новый репозиторий, также генерируется .gitignore. 


Поддерживаются следующие команды:
hdl.py -git clone      = клонировать весь hdl репо;
hdl.py -git pull       = заапдейтить и смержить имеющиеся репозитории;
hdl.py -git cmd        = запустить гит консоль в текущей директории;
kungfu.py* -git upload = загрузить текущий репо на webdav;
  kungfu.py - любой скрипт в котором определен hdlManager;
  если локального репо нет - будет создан новый;
